Settings

  • Discogs API Token

    Required for vinyl barcode and title lookups. Get a free token at discogs.com/settings/developers.

  • TMDB API Key

    Optional. Enables DVD title autocomplete with covers, year, and genre. Get a free key at themoviedb.org/settings/api (sign up, then request an API key, "Developer" type, takes ~1 minute).

  • Cloud Auto-Backup

    Sign in once. App silently overwrites the_j_stash.json in your cloud drive on every save, edit, and delete. Restore in one tap if your device storage is ever wiped.

    Not connected.

    Google Drive

    Paste your Google OAuth Client ID and Client Secret (both required by Google's token endpoint even with PKCE).

    OneDrive

    Paste a Microsoft Application (client) ID.

    GitHub

    Stores backups as commits in a repo. Use a fine-grained Personal Access Token scoped to one repo with Contents: Read & Write.

  • Export Backup

    Saves files to your Downloads folder.

  • Import Backup

    Restore from a JSON backup. Existing entries with the same ID are replaced.

  • Erase All Data

    Permanently deletes every entry in your local database. This cannot be undone.

  • About

    The J Stash runs entirely on your device. No accounts, no servers. Your data lives in browser storage; backups are in your control. Add to home screen for full app experience.

Set Up The J Stash

Your data lives in a private GitHub repo. This way it survives anything Brave does to local storage.

You'll need:

  • A private GitHub repo for your data (separate from the app)
  • A fine-grained Personal Access Token with Contents read/write access to that repo

No data yet? Just create an empty private repo — the app will populate it.

Paste your GitHub access info:

Token must have Contents: Read & Write for this repo. Get one at github.com/settings/tokens?type=beta

Connection looks good. Optionally add API tokens for richer lookups (you can skip and add later from Settings → Advanced):

Tokens save to your config file in GitHub. They persist across Brave wipes.
